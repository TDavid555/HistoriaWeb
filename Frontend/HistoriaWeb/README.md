HistóriaWeb

Indítás

1. Node.js telepítése

Töltsd le és telepítsd a Node.js-t innen: https://nodejs.org
Válaszd az LTS verziót.

2. Csomagok telepítése

Nyiss egy parancssort (CMD) a projekt mappájában, majd lépj be a HistoriaWeb mappába:

    cd Frontend/HistoriaWeb

Ezután futtasd le:

    npm install

Ez letölti a szükséges fájlokat. Eltarthat 1-2 percig.

3. Az alkalmazás indítása

    npm start

4. Megnyitás böngészőben

Az alkalmazás az alábbi címen érhető el:

    http://localhost:4200
